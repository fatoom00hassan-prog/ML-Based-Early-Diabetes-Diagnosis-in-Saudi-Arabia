# ML-Based Early Diabetes Diagnosis in Saudi Arabia

A systematic review exploring how Machine Learning (ML) techniques can support the **early diagnosis of diabetes**, with a focus on studies involving **Saudi Arabian and Gulf-region populations (2020–2024).**

---

## Project Overview

Diabetes Mellitus is one of the most prevalent chronic diseases worldwide, and its prevalence is particularly high in Saudi Arabia. Early diagnosis is essential to prevent complications and improve patient outcomes.

This project presents a **systematic review of machine learning approaches used for early diabetes detection**, analysing recent studies and identifying the most effective algorithms, predictive features, and research gaps.



---
## Methodology

This research follows the PRISMA systematic review methodology to analyse machine learning approaches used for early diabetes diagnosis.



## Machine Learning Models Reviewed
The reviewed studies used several ML algorithms, including:

- Random Forest (RF)
- Support Vector Machine (SVM)
- Decision Tree (DT)
- Gradient Boosting (GB)
- XGBoost
- K-Nearest Neighbors (KNN)
- Artificial Neural Networks (ANN)
- Ensemble Models

## Key Findings

The main findings of the review include:

- Ensemble ML models outperform traditional statistical approaches.
- Random Forest, SVM, and XGBoost achieve the best predictive performance.
- Models trained on **regional datasets** show higher clinical relevance.
- Explainable AI techniques improve model interpretability.
- There is still a lack of **large Saudi-specific medical datasets**.

---

## Research Gap

Existing studies show strong ML performance for diabetes prediction, but there is still a lack of large Saudi-specific datasets and standardized evaluation methods across studies.
